---
title: Histogram
description: Create histograms from data with adjustable bin width
---

```{anywidget} ../../widgets/histogram/dist/widget.mjs
{
  "data": {
    "cities": "../../public-data/cities.json",
    "ccv": "../../public-data/ccv.json",
    "gmat": "../../public-data/gmat.json"
  },
  "xLabel": "Value",
  "bins": 10,
  "show_normal": false
}
```

You can change the inputs to highlight different parts of the histogram; the area highlighted is given in the lower left corner.
