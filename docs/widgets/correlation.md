```{anywidget} ../../widgets/correlation/dist/widget.mjs
{
  "r": 0.7,
  "n": 50,
  "show_sd_lines": true,
  "show_sd_line": false,
  "show_regression": true,
  "include_added_points": true,
  "seed": 42
}
```

Play with the sliders to change the correlation coefficient and the number of data. The buttons add lines at the point of averages ±1 SD, the SD line and the regression line. You can add points by clicking in the scatterplot. Whether those points are used in the computations is controlled by another button. The last button clears the points you added by clicking.
